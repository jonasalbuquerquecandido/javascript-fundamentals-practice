# Principais Fórmulas para Programadores

## 🔑 Fórmulas para decorar primeiro

### 1. Operações básicas
- Soma: a + b
- Subtração: a - b
- Multiplicação: a * b
- Divisão: a / b
- Módulo: a % b → resto da divisão

### 2. Fatorial
n! = n × (n-1) × (n-2) × ... × 1  
- n = número inteiro positivo  
- ! = símbolo de fatorial  

### 3. Fibonacci
F(n) = F(n-1) + F(n-2)  
- Sequência: 0, 1, 1, 2, 3, 5, 8...  

### 4. Média aritmética
média = soma dos valores / quantidade  

### 5. Pitágoras
c² = a² + b²  
- c = hipotenusa  
- a, b = catetos  

### 6. Potenciação
a^b = a multiplicado por ele mesmo b vezes  

---

## 📘 Fórmulas para consultar quando precisar

### Combinações
C(n,k) = n! / (k!(n-k)!)  

### Logaritmo
log_a(b) → potência que a deve ter para resultar em b  

### Média ponderada
Σ(valor × peso) / Σ(peso)  

### Probabilidade
P = casos favoráveis / casos possíveis  

### Trigonometria básica
sen, cos, tan → relações entre lados de triângulos
